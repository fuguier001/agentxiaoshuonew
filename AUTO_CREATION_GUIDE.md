## 🎯 快速开始

### 步骤 1: 打开全自动创作页面

访问：http://localhost:5173  
点击左侧菜单：**🤖 全自动创作**

### 步骤 2: 填写基本信息

只需要填写 3 个字段：

1. **书名**: 例如《都市修仙》
2. **类型**: 选择 玄幻/仙侠/都市/历史/科幻/游戏
3. **简介**: 一句话描述核心设定

**示例**:
```
书名：《绝世武神》
类型：玄幻
简介：一个普通少年意外获得上古武神传承，从此踏上修炼之路，战天骄，灭强敌，最终登临武道巅峰
```

### 步骤 3: 点击"开始全自动创作"

点击按钮后，AI 会自动：

```
🌍 生成世界观地图 (30 秒)
   - 修炼体系
   - 地理环境
   - 势力分布
   - 资源分布
   - 谜团埋设

📋 生成 3000 章规划 (60 秒)
   - 15 卷详细规划
   - 节奏控制点
   - 爽点分布
   - 地图切换

👥 生成人物体系 (30 秒)
   - 主角深度设定
   - 主要配角
   - 反派角色
   - 人物关系网

🎣 生成伏笔网络 (30 秒)
   - 短期伏笔
   - 中期伏笔
   - 长期伏笔
   - 终极伏笔

✍️ 创作第一章 (30 秒)
   - 3000 字正文
   - 专业四段式结构
   - 包含爽点和悬念
```

**总耗时**: 约 2-5 分钟

## 🎨 专业特性

### 1. 世界观地图（俄罗斯套娃结构）

```
大世界
├── 东荒大陆
│   ├── 大周皇朝
│   │   ├── 青云城
│   │   └── 落云城
│   └── 大夏皇朝
└── 西漠大陆
    └── ...
```

### 2. 3000 章规划（节奏精准）

```
第 1-10 章：开篇铺垫，第一个爽点
第 11-20 章：小高潮，打脸反派
第 21-50 章：发展，积累实力
第 51-100 章：中高潮，宗门大比
...
每 100 章：一个大高潮
每 300 章：换一个大地图
```

### 3. 爽点分布（科学安排）

- **打脸**: 每 5 章一次
- **升级**: 每 20 章一次
- **美人**: 每 30 章一次
- **收获**: 每次战斗后

### 4. 伏笔网络（四层设计）

- **短期**: 1-10 章回收，保持短期期待
- **中期**: 10-50 章回收，推动篇章剧情
- **长期**: 100-500 章回收，支撑主线
- **终极**: 全书回收，支撑核心主题

## 🎯 使用建议

### 1. 简介怎么写？

**好的简介**:
```
一个普通大学生意外获得修仙传承，在都市中修炼成长，
扮猪吃虎，打脸富二代，最终登临巅峰
```

**不好的简介**:
```
一个关于修仙的故事
```

### 2. 类型选择

- **玄幻**: 修炼、武道、异能
- **仙侠**: 修仙、法宝、宗门
- **都市**: 现代、职场、商战
- **历史**: 古代、穿越、权谋
- **科幻**: 未来、科技、星际
- **游戏**: 网游、电竞、虚拟世界

### 3. 创作后做什么？

1. **查看蓝图**: 检查世界观和规划
2. **修改调整**: 可以手动调整设定
3. **继续创作**: AI 继续生成后续章节
4. **导出保存**: 导出为 TXT/EPUB

## ⚠️ 注意事项

### 1. 创作时间
- 第一次创作：2-5 分钟
- 后续章节：30 秒/章
- 请耐心等待，不要关闭页面

### 2. LLM 配置
- 确保已配置 API Key
- 推荐使用 eggfans
- 检查网络连接

### 3. 内容质量
- AI 生成内容仅供参考
- 建议人工审核和调整
- 可以多次生成选择最佳
