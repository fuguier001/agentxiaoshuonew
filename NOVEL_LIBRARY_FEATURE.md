### 5. 一键下载功能

**下载格式**:
1. **TXT 文本** (.txt)
   - 纯文本格式
   - 适合手机阅读
   - 兼容性好

2. **Markdown** (.md)
   - 带格式标记
   - 适合发布到网站
   - 支持标题/粗体等

3. **JSON (含设定)** (.json)
   - 包含完整设定
   - 可选包含：
     - 世界观地图
     - 剧情规划
     - 人物设定
     - 伏笔网络
   - 适合备份和迁移

**下载范围**:
- 全部章节
- 选择章节

## 🎯 使用流程

### 查看小说
1. 进入小说仓库
2. 点击"📖 查看"按钮
3. 查看小说信息和章节列表
4. 点击章节可以查看内容

### 编辑小说
1. 点击"✏️ 编辑"按钮
2. 修改书名/简介/类型等
3. 点击"保存"

### 编辑章节
1. 查看小说详情
2. 点击章节的"编辑"按钮
3. 在文本框中修改内容
4. 支持 Markdown 预览
5. 点击"保存"

### 下载小说
1. 点击"📥 下载"按钮
2. 选择下载格式 (TXT/Markdown/JSON)
3. 选择下载范围 (全部/选择)
4. 如果选 JSON，可以选择包含的设定
5. 点击"下载"
6. 自动保存到本地

### 续写小说
1. 点击"✍️ 续写"按钮
2. 自动跳转到写作面板
3. 选择或创建新章节
4. 继续创作

## 🔧 技术实现

### 前端组件
- `NovelLibrary.vue` - 小说仓库主页面
- 搜索/筛选功能
- 卡片式布局
- 对话框管理

### 下载功能
```javascript
// TXT 下载
const downloadAsTxt = (chapters) => {
  const content = chapters.map(ch => 
    `第${ch.chapter_num}章 ${ch.title}\n\n${ch.content}\n\n`
  ).join('---\n\n')
  
  const blob = new Blob([content], { type: 'text/plain;charset=utf-8' })
  // 触发下载
}
```

### API 调用
- `apiClient.novels.list()` - 获取小说列表
- `apiClient.novels.get(id)` - 获取小说详情
- `apiClient.novels.update(id, data)` - 更新小说
- `apiClient.novels.getChapters(id)` - 获取章节列表
- `apiClient.novels.updateChapter(...)` - 更新章节
