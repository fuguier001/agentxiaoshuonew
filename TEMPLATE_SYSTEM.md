## 📖 模板分类

### 1. 大纲模板 (5 种)

#### 🎬 三幕式结构 (three_act)
**适用**: 商业小说、科幻小说、冒险小说  
**特点**: 
- 经典好莱坞结构
- 节奏明确 (25%-50%-25%)
- 适合新手

**生成内容**:
- 第一幕：铺垫 (开场画面、激励事件、第一转折点)
- 第二幕：对抗 (副线、游戏时间、中点、反派逼近、一无所有)
- 第三幕：解决 (灵魂黑夜、高潮、结局)

---

#### 🗡️ 英雄之旅 (heros_journey)
**适用**: 奇幻小说、冒险小说、成长小说  
**特点**:
- 约瑟夫·坎贝尔经典理论
- 12 个阶段
- 强调内在成长

**生成内容**:
- 出发 (平凡世界、冒险召唤、拒绝召唤、遇见导师、跨越门槛)
- 启蒙 (考验、接近洞穴、严峻考验、获得宝剑)
- 回归 (返回的路、复活、带着万能药回归)

---

#### 📜 起承转合 (qichengzhuanhe) ⭐ 推荐
**适用**: 武侠小说、言情小说、历史小说  
**特点**:
- 中国传统叙事结构
- 符合中文阅读习惯
- 注重意境和留白

**生成内容**:
- 起 (背景交代、人物登场、矛盾萌芽)
- 承 (情节发展、人物关系、冲突升级)
- 转 (重大转折、矛盾爆发、人物抉择、情感高潮)
- 合 (冲突解决、人物归宿、主题升华、余韵)

---

#### 🔀 多线叙事 (multi_thread)
**适用**: 悬疑小说、群像小说、宏大题材  
**特点**:
- 多线并行
- 线索交织
- 最终汇聚

**生成内容**:
- A 线 (主线) 设计
- B 线 (副线 1) 设计
- C 线 (副线 2) 设计
- 线索交织点 (3-4 次)
- 最终汇聚

---

#### 📺 单元剧结构 (unit_drama)
**适用**: 侦探小说、灵异小说、都市传说  
**特点**:
- 独立单元 + 主线贯穿
- 每章可独立阅读
- 适合长篇连载

**生成内容**:
- 主线设定 (核心谜团、主角成长、最终真相)
- 8-12 个单元设计
- 单元节奏 (开篇、发展、转折、高潮、收尾)

---

### 2. 人物模板 (3 种)

#### 👤 主角模板 (protagonist) ⭐ 深度
**适用**: 第一主角设计  
**生成内容** (1500-2000 字):
- 基础信息
- 外貌特征 (整体、面部、身材、标志、穿着)
- 性格层次 (表层、中层、深层、缺陷、优势)
- 背景故事 (童年、转折点、创伤、成长)
- 内在驱动 (核心欲望、核心恐惧、核心信念、内在冲突)
- 人物弧线 (开始、成长、最终、变化)
- 能力设定
- 关系网络
- 语言特征 (口头禅、风格、经典台词)

---

#### 👿 反派模板 (antagonist)
**适用**: 主要反派设计  
**特点**: 避免脸谱化，塑造有深度的反派

**生成内容** (1500-2000 字):
- 基础信息
- 外在形象
- 核心动机 (表面目标、真实目标、动机来源、合理性)
- 背景故事 (成长、创伤、转折、黑化)
- 性格特点 (优点、缺点、执念、底线)
- 与主角关系 (镜像、共同点、分歧、对决意义)
- 能力与资源
- 经典场景 (高光、失败、救赎)
- 经典台词

---

#### 🎭 配角模板 (supporting)
**适用**: 配角群像设计  
**特点**: 功能性明确，有记忆点

**生成内容**:
- 功能定位
- 性格特点 (2-3 关键词)
- 记忆点
- 与主角关系
- 个人故事
- 经典台词

---

### 3. 章节模板 (3 种)

#### 📖 开篇章节模板 (opening)
**适用**: 第 1-3 章  
**目标**: 吸引读者，建立基调

**生成内容** (1500-2000 字):
- 核心目标 (吸引读者、建立基调、引入主角、埋下伏笔)
- 开场场景 (500 字)
- 激励事件 (300 字)
- 主角反应 (400 字)
- 章节结尾 (300 字，悬念)
- 写作要点

---

#### 📈 发展章节模板 (development)
**适用**: 故事发展阶段  
**目标**: 推进剧情，人物发展

**生成内容** (1500-2000 字):
- 本章定位
- 开场 (200 字)
- 发展部分 A (400 字)
- 发展部分 B (400 字)
- 小高潮 (300 字)
- 结尾 (200 字)
- 伏笔与呼应

---

#### 🔥 高潮章节模板 (climax)
**适用**: 高潮章节  
**目标**: 情感爆发，让读者热血沸腾

**生成内容** (2000-2500 字):
- 高潮定位
- 开场 (200 字，暴风雨前的宁静)
- 对抗升级 (500 字)
- 转折点 (300 字)
- 最终对决 (600 字)
- 余波 (300 字)
- 情感设计

---

### 4. 情节模板 (2 种)

#### ⚔️ 冲突升级模板 (conflict_escalation)
**适用**: 需要紧张感的章节  
**生成内容**:
- 初始状态
- 第一次升级 (小冲突)
- 第二次升级 (中冲突)
- 第三次升级 (大冲突)
- 高潮对决
- 余波

---

#### ❓ 悬念设置模板 (suspense)
**适用**: 悬疑、推理章节  
**生成内容**:
- 核心谜团
- 信息控制 (告诉、隐藏、误导)
- 悬念层次 (表层、中层、核心)
- 悬念推进 (5 次提示)
- 节奏控制 (张弛有度)

## 🎯 如何使用

### API 调用

#### 1. 获取模板列表
```bash
GET /api/ai/templates

返回:
{
  "outline": [...],
  "character": [...],
  "chapter": [...],
  "plot": [...]
}
```

#### 2. 使用模板生成大纲
```bash
POST /api/ai/generate-outline
{
  "title": "我的小说",
  "genre": "fantasy",
  "description": "简介...",
  "template_id": "qichengzhuanhe"  // 选择模板
}
```

**可选 template_id**:
- `three_act` - 三幕式
- `heros_journey` - 英雄之旅
- `qichengzhuanhe` - 起承转合 ⭐ 推荐
- `multi_thread` - 多线叙事
- `unit_drama` - 单元剧

#### 3. 使用模板生成人物
```bash
POST /api/ai/generate-characters
{
  "title": "我的小说",
  "genre": "wuxia",
  "outline": "...",
  "template_id": "protagonist",  // 主角模板
  "count": 1
}
```

#### 4. 使用模板生成章节大纲
```bash
POST /api/ai/generate-chapter-outline
{
  "novel_title": "我的小说",
  "chapter_num": 1,
  "overall_outline": "...",
  "template_id": "opening"  // 开篇模板
}
```
