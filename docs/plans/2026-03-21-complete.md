## 🚀 快速启动指南

### 1. 环境准备

```bash
# Python 3.10+
python --version

# Node.js 18+
node --version

# Redis
redis-server --version
```

### 2. 安装依赖

```bash
cd novel-agent-system

# 后端
pip install -r requirements.txt

# 前端
cd frontend
npm install
```

### 3. 配置 LLM

编辑 `backend/config/llm_providers.json`:

```json
{
  "default_provider": "volcengine",
  "providers": {
    "volcengine": {
      "api_format": "openai",
      "api_key": "你的火山引擎 API Key",
      "base_url": "https://ark.cn-beijing.volces.com/api/v3",
      "model": "doubao-pro-32k",
      "timeout": 60,
      "enabled": true
    }
  }
}
```

### 4. 启动服务

```bash
# 终端 1 - 后端
cd backend
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# 终端 2 - Redis
redis-server

# 终端 3 - 前端
cd frontend
npm run dev
```

### 5. 访问系统

- **前端**: http://localhost:5173
- **后端 API**: http://localhost:8000
- **API 文档**: http://localhost:8000/docs
- **健康检查**: http://localhost:8000/api/health

## 💡 使用示例

### 1. 分析作品学习风格

```python
from app.agents.registry import get_agent

# 获取学习分析师 Agent
learning_agent = get_agent('learning_agent')

# 分析作品
result = await learning_agent.execute({
    'action': 'analyze_external_work',
    'author': '金庸',
    'title': '天龙八部',
    'text': '小说内容...'
})

print(f"分析完成：{result['analysis_id']}")
print(f"风格特征：{result['analysis']['key_features']}")
```

### 2. 创作章节

```python
from app.tasks.agent_tasks import start_chapter_workflow

# 启动章节创作工作流
workflow = start_chapter_workflow(
    chapter_num=5,
    project_id='project_001',
    outline='主角首次遭遇反派，展现两人性格对比'
)

print(f"工作流 ID: {workflow['workflow_id']}")
print(f"状态：{workflow['status']}")
```

### 3. 查询工作流状态

```python
from app.tasks.agent_tasks import get_workflow_status

status = await get_workflow_status('wf_001')

print(f"进度：{status['progress']}%")
print(f"当前阶段：{status['current_stage']}")
```

### 4. 保存章节到文件

```python
from app.storage.file_manager import get_file_manager

fm = get_file_manager('./projects/test')

await fm.save_chapter(
    chapter_num=5,
    content='章节正文...',
    title='初遇反派',
    metadata={
        'style_id': 'style_001',
        'techniques': ['多线交织', '反差塑造']
    }
)
```
